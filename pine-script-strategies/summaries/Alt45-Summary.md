## ðŸ”¬ Dual-Momentum Confirmation Logic

### Momentum Components:

**1. Price Momentum (Donchian Breakout):**
```
Current price > highest price in last 20 periods
Confirms absolute price strength
```

**2. Relative Strength:**
```
Stock performance vs sector ETF:
  QQQ vs SPY
  UNH vs XLV
  CAT vs XLI
  GOOGL vs XLK

Relative Strength = (Stock return / Sector return) - 1
Positive = outperforming, Negative = underperforming
```

**3. Dual Confirmation Required:**
```
Entry requires:
  Price Momentum = TRUE (Donchian high)
  AND
  Relative Strength > 0 (outperforming sector)

Exit if:
  Price Momentum = FALSE (stop loss hit)
  OR
  Relative Strength < 0 (underperforming sector)
```
