## üéØ What This Strategy Does

Alt26 uses **fractional pyramiding** to add to winning positions:

1. **100% initial position** - Enter on first Donchian breakout
2. **Add 75%** - Add to position on next breakout
3. **Add 50%** - Add again on continued breakout
4. **Add 25%** - Final add if trend continues
5. **Exit ALL** - Exit entire position on stop loss

### Total Position: 250% of Base Size
If you normally trade 10 contracts:
- Entry: 10 contracts
- Add #1: 7.5 contracts (total 17.5)
- Add #2: 5 contracts (total 22.5)
- Add #3: 2.5 contracts (total 25 = 250%)

## üí° Pro Tips

### Pyramiding Rules:
- **Only pyramid winners** - Never add to losing trades
- **Use profits to fund adds** - Don't risk fresh capital
- **Exit everything together** - Don't try to scale out
- **Requires strong trend** - Weak trends will whipsaw

### Risk Management:
- Initial position: Risk 2% of account
- Each add: Funded from profits (not new risk)

**Pyramid Adds:**

| Add # | Size | Contracts | Total Contracts | Funded By |
|-------|------|-----------|-----------------|-----------|

| Add 3 | 25% | 2-3 | 24-26 | Profits from add 2 |
